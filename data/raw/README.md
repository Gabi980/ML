# Raw data

Place the Kaggle CSV files here before running the training pipeline.

Expected filenames:

- `kc_house_data.csv` for House Sales in King County, USA
- `WA_Fn-UseC_-Telco-Customer-Churn.csv` for Telco Customer Churn

The training script also tries to detect alternative CSV filenames that contain
`house` or `churn`.
